Create/Verify Creation of ITGlue Site

This will mostly be a verification, if no site is set up get with TS/Lead Project Manager to get site created.

Create/Verify QuickPass Password Rotation

  1. Select “Add Account” in the top right corner.
  2. Set to Manual.
  3. Select Active Directory.
  4. Select OU you need.
  5. Search HTADMIN.
  6. Check and add (it should autofill password, if not find credentials in ITGlue and put in password).
  7. SAVE.
  8. Hit switch for password rotation to start.

5AM-6AM is when rotation happens.

How to install the Agent

Create ArmorPoint Site

Fenoglio will create the site for now. Keep track of the number of endpoints coming in and push install scripts when possible.

Create Datto SAAS Site

  1. Select Status Tab at the top of the page.
  2. SaaS Protection Status.
  3. Create Client.
  4. Configure with month-to-month infinite settings.
  5. Leave all other defaults.
  6. Be sure the client is set to NO license caps.
  7. Agree and Add SaaS Protection Client.

Will require credentials for O365 from ITGlue.

Once set up, the client will take a few days to completely ingest services, be patient and expect a few days of false errors until the ingesting period is complete.
