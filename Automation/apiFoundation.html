Ninja API Script Setup

API scripts are run on noc-mgmt (temp).

It has the device role "NOC Workstation" which has role-based custom fields assigned to it.

Those custom fields are "ClientID" and "Client secret", both are security type.

API scripts run on the machine will decrypt those two fields and use them to connect to NinjaOne's API to pull info from our instance of NinjaOne.

The credentials are wiped right after use on the machine.

The scripts utilize PowerShell v7 and also Luke Whitelock's NinjaOne Documentation PowerShell module.

can be found at https://docs.mspp.io/.
