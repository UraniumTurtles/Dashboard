Ninja Maintenance Mode: High Touch ITS SOP

Purpose

This document will detail the High Touch procedure for setting a Maintenance window for a Ninja device or set of devices. This can include servers or network devices.

Reasons for a Maintenance window:

  • Planned device maintenance which would cause that device to lose network connectivity or generate OS errors.
  • Known upcoming power or network outage.
  • Replacing or removing hardware.
  • Decommissioning of devices (in those cases we want to shut a server down for example for a few weeks before completely removing).

Ninja help documentation for Maintenance Mode (for details on how to set Maintenance on a device or set of devices):

Maintenance Mode

High Touch Specific Procedures to Follow:

  • Only suppress Condition based alerting when setting the maintenance window.
  • There will be exceptions where we want to disable other features such as scripting but generally, we only want to suppress alerting during maintenance.
  • Leaving the other boxes unchecked allows us to still patch, protect with AV, and use scripts on this device if needed.
  • Do not set the End Date & Time on a Maintenance window to Never. Please assign a safe date and time in the future when we want alerting to resume on the device.
  • Leaving a maintenance window wide open with no end date is not necessary in nearly every case. This will just invite scenarios where we need alerting on the device and the maintenance window never ends because a date was not set.
  • This includes server decommissions. We should have an idea on when we want to remove an old server completely so, please set the date to that window. If we are unsure how long a server may be left shutdown, please pick a date safely in the future.
  • This at least allows us to review why a server is shutdown should that window expire, and an alert be triggered. This can be a helpful reminder we need to remove a decommissioned server vs. leaving it with no end date and the server ends up being in Ninja for months unnecessarily.
